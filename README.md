# Estrous Cycle Modulation of Head Direction Coding Properties 

**Purpose**: Provide code resources using the Pynapple package to analyze common head direction cell coding properties across the rodent estrous cycle. 

Head Direction Coding Properites Analzyed: 
1) Maximum Firing Rate During Exploration
2) Average Firing Rate During Sleep
3) Tuning Width
4) Continuous Stability
5) Interlevered Stability
